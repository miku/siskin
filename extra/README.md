## DOI and DDC

There are no DDC in crossref.

Extract DOI and subjects from DOAJ dump.

```shell
$ taskcat DOAJDownloadDump | \
    jq -rc '{"doi": .bibjson.identifier[] | select(.type == "doi").id, "s": .bibjson.subject}'
```
