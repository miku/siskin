siskin
======

Various tasks for metadata handling.

[![pypi version](http://img.shields.io/pypi/v/siskin.svg?style=flat)](https://pypi.python.org/pypi/siskin)

Installation
------------

Python 2.7 required.

    $ pip install --upgrade siskin

Run
---

List tasks:

    $ tasknames

Run task:

    $ taskdo AIUpdate
